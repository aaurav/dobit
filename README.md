# Dobit

**Dobit** is a simple Rust CLI tool for performing bitwise operations on numbers. It supports:

- Unary operations: `not`
- Binary operations: `xor`, `and`, `or`
- Shifts and rotations: `shl`, `shr`, `rol`, `ror`

## Installation

Clone the repository and build the project using Cargo:

```bash
git clone https://github.com/<your-username>/dobit.git
cd dobit
cargo build --release
